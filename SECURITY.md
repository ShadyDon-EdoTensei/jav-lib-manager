# Security Policy

## Supported Versions

最新版本的 `main` 分支会获得安全更新支持。

| Version | Supported          |
| ------- | ------------------- |
| latest  | :white_check_mark: |

## Reporting a Vulnerability

**请不要**公开报告安全问题。

如果你发现了安全漏洞，请私下联系我们：

- **邮箱**: ShadyDon-EdoTensein@users.noreply.github.com
- **GitHub Security**: [使用 GitHub Security Advisories](https://github.com/ShadyDon-EdoTensei/jav-lib-manager/security/advisories)

请提供以下信息：

1. 漏洞描述
2. 复现步骤
3. 潜在影响
4. 如果已知，建议的修复方案

我们会在 72 小时内确认收到你的报告，并提供后续更新计划。

一旦漏洞被确认，我们会：

1. 确认漏洞并评估严重程度
2. 制定修复计划
3. 在合理的时间内发布修复
4. 通知用户进行更新

## Security Best Practices

作为用户，你可以：

1. **保持更新**: 及时更新到最新版本
2. **审计依赖**: 定期审查 `requirements.txt` 中的依赖包
3. **安全扫描**: 使用安全扫描工具检查代码
4. **报告问题**: 发现安全问题及时报告

感谢你帮助保持 JAV Lib Manager 的安全！
